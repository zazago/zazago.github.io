CocoaPods用上了!!!

Jul 25th, 2013 | Comments

前言

知道有CocoaPods这东西已经有很长时间了,之前装过一两次，但是gem install cocoapods总是下载失败出现error，貌似是网络原因，CocoaPods的官网都上不去。

昨晚在iosfeed看到了一个BeyondVincent分享的插件CocoaPods Xcode Plugin(仅支持Xcode5)，遂又尝试的安装了一次CocoaPods，总算成功！

CocoaPods安装使用

打开终端，使用命令

 gem install cocoapods
 pod setup

等待CocoaPods下载安装完成。

下载CocoaPods Xcode Plugin插件，运行后重启Xcode5即可使用插件。

用Xcode5打开项目，在Product里找到CocoaPods，点击Create Podfile创建Podfile文件.

如下输入platform、支持的最低版本以及所要集成的开源项目。

platform :ios, '5.0'

pod "MBProgressHUD"
pod "FMDB"
pod "MMDrawerController"
pod "TPKeyboardAvoiding"

保存后，点击install Pods in Podfile就会自动创建Pods项目。如果没有自动创建也可以使用终端，到所要执行的项目目录下使用pod install命令创建Pods项目。

此外，该插件还集成了UpDate CocoaPods的功能，方便使用者对CocoaPods进行更新。感谢作者！

Balsamiq Mockups工具初体验

Jul 17th, 2013 | Comments

Balsamiq Mockups简介

原型开发软件，可以进行软件草图的绘制。

Balsamiq Mockups初体验

软件界面


该软件分为三个区块，第一个区块是包含撤销、复制、黏贴等操作的区域；第二个区域是控件区域，在里面可以找到大部分的控件、图形进行原型的绘画(直接拖动至绘画区或双击即可)，基本上还是能满足草图的绘画要求的；第三个区域是绘画区，你可以在里面进行绘画，如大小颜色等的修改，文字的修改等。

随手玩一下

随便拖几个控件进行绘画，就拿iphone这个控件举例，点击控件会出现编辑框

在编辑框里面可以进行iphone4/5的选择大小,背景，电池栏等的设置，为了防止无意间拖动控件，也可以点击锁按钮锁住控件(右键点击控件选择解锁项即可解锁)。具体一些操作还是在实践中进行摸索，基本都不难。

导出项目

导出绘制的图片可以选择导出为png或者pdf，当绘制内容很多的时候导出成pdf还是很方便的。

结束语

使用Mockups来绘制草图还是很方便的，提供的控件库也基本能满足需要，如果在以后工作中碰到例如绘制草图这样的工作任务，Mockups应该会成为我的选择。之前也有接触另一款应用POP，POP需要对所绘制的草图进行拍摄，然后再图片上增加交互动作就可以进行交互，也许这两款工具结合使用能收到不错的效果。

最近有辞职的想法，一来是因为leader的管理方式以及沟通方法，二来是团队给个人的感觉，工作带给个人的感觉。因此，和leader有了一次漫长的对话，谈的很杂，聊了很多，其中谈的更多的是责任感，担当以及感恩。不可否认，我是抱有一个感恩的心态的，任何一个团队对于一个实习生的培养都是不容易的一件事情，这点我是明白的。担当以及责任感，确实无论在何处在何方，这两者都是一个想要获得成功的人应该具有的品质，个人的信誉或者说给人的印象就是建立在是否有责任感是否有担当。我自己从我自己的行为上考量，确实也有些欠妥的地方。最后，当然就是再考虑考虑。希望自己努力的为公司做更多的事情，同时当然也要提高自己的专业水平。至少在未来的某一天，能够理直气壮地说，我是一个有责任感有担当的人。
