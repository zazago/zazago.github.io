CocoaPods用上了!!!

Jul 25th, 2013 | Comments

前言

知道有CocoaPods这东西已经有很长时间了,之前装过一两次，但是gem install cocoapods总是下载失败出现error，貌似是网络原因，CocoaPods的官网都上不去。

昨晚在iosfeed看到了一个BeyondVincent分享的插件CocoaPods Xcode Plugin(仅支持Xcode5)，遂又尝试的安装了一次CocoaPods，总算成功！

CocoaPods安装使用

打开终端，使用命令

 gem install cocoapods
 pod setup

等待CocoaPods下载安装完成。

下载CocoaPods Xcode Plugin插件，运行后重启Xcode5即可使用插件。

用Xcode5打开项目，在Product里找到CocoaPods，点击Create Podfile创建Podfile文件.

如下输入platform、支持的最低版本以及所要集成的开源项目。

platform :ios, '5.0'

pod "MBProgressHUD"
pod "FMDB"
pod "MMDrawerController"
pod "TPKeyboardAvoiding"

保存后，点击install Pods in Podfile就会自动创建Pods项目。如果没有自动创建也可以使用终端，到所要执行的项目目录下使用pod install命令创建Pods项目。

此外，该插件还集成了UpDate CocoaPods的功能，方便使用者对CocoaPods进行更新。感谢作者！
